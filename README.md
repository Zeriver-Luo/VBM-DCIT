# VBM-DCIT
Vision-Based-Measurement-of-Dust-Concentration-by-Image-Transmission

该源代码为"Vision-Based Measurement of Dust Concentration by Image Transmission"论文的实现，我们感谢您的意见/建议,有关代码的问题，请发送至GuohuiLi@sicnu.edu.cn.

The source code is the implementation of "Vision-Based Measurement of Dust Concentration by Image Transmission" paper.We appreciate any comments/suggestions. Questions regarding the code can be directed to GuohuiLi@sicnu.edu.cn.
*********************************************************************************************************************************************************************************


The code runs on Windows 7 with MATLAB R2015b.

To get a quick overview:

1.打开demo.m编辑BackGround_img的路径,图片格式类型。
2.设置拟合方式以及拟合粉尘浓度。
3.设置拟合图片的路径与图片格式类型。
4.设置测试图片的完整路径。
5.Run demo.m.你会得到拟合参数与测试粉尘浓度值。

1.	Opens the path to the demo.m edit BackGround_img, image format type.
2.	Set fit mode and fit dust concentration
3.	Set the path and format type of fit image
4.	Set the full path to the test image
5.	Run demo.m. You will get fitting parameters and test dust concentration values.
